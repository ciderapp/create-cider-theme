<img src="https://img.shields.io/github/package-json/v/ciderapp/create-cider-theme">

## Usage
npm
```bash
npm install
npm run build
npx create-cider-theme
```
yarn
```bash
yarn install
npm build
npx create-cider-theme
```
pnpm (Recommended)
```bash
pnpm install
pnpm build
npx create-cider-theme
```
